VH e VW

Viewport Width

É uma unidade de medida relativa. Utiliza a largura do viewport, ou seja, dependendo do tamanho do dispositivo utilizado pelo usuário, o elemento terá um tamanho diferente, compatível com o dispositivo utilizado. Não é bom utiliza-lo para estilizar o tamanho de textos.

width: 100vw;
width: 100%;

Viewport Height

Também é uma unidade de medida relativa. E utiliza a altura do viewport, que também dependendo do tamanho da tela do dispositivo, será exibido em um tamanho diferente, se adequando ao tamanho da tela utilizada.

height: 100vh;
height: 100%;
